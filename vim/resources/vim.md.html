 4 Ex command line mode <!-- {{{1 -->
 5 ====================
 6 General ex command line <!-- {{{2 -->
 7 -----------------------
 8 - `set {option1} {option2}` : issue two set commands (same for setlocal)
 9     - `set cursorline! hlsearch!`
10 - `{command1} | {command2}` : issue two ex commands in one line
11     - `echo 'hello' | set ruler!`
12 - `exe {string}` : run the string as an ex command line
13     - `exe 'set colorcolumn='.(40+20)`
14 - `redraw!` : sometimes the vim rendering is corrupted, run this will force vim to redraw
15 - `q:` : opens command history window
16 - `put =expression` : evaluate the expression, paste the result and enter normal mode
17     - `put =1+2`
18 
19 Editing in ex command line <!-- {{{2 -->
20 --------------------------
21 - `<c-r><c-a>` : Pull <CWORD> onto search/command line
22 - `<c-r><c-w>` : Pull <cword> onto search/command line
23 - `<c-r>{register}` : Pull the content of {register} onto search/command line
24     - `<c-r>/` : last search
25     - `<c-r>:` : last ex command
26     - `<c-r>@` : last yanked
27     - `<c-r>%` : file name
28 
29 Vim-shell interactions <!-- {{{2 -->
30 ----------------------
31 - `sh` : open a temporary shell, return to vim upon its exit
32 
33 Buffer management <!-- {{{2 -->
34 -----------------
35 - `badd {file_name}` : add {file-name} to the buffer list without opening it
36     - `badd resources/a`
37 
38 File operation <!-- {{{2 -->
39 --------------
40 - `:edit!` / `:e!` : reload current buffer from disk
41 - `:E` / `:e.` / `:e .` : open the current directory in netrw
42 - `X` : crypt the current file
43     - `e resouces/encryptme`
44     - write something
45     - `X`
46 - `:find {pattern}` : open the file that matches pattern, if multiple files are present it fails
47   with message **E77: Too many file names**
48     - `find **/*foundm*`
49 - `enew` : open a new unnamed buffer
50 - `vnew` : open a new unnamed buffer in the vertical split
51 - `w {file}` / `write {file}` : save the current buffer to file and remind in the current file
52     - `write! resources/write`
53         - we are still at vim.md
54 - `w {file}` / `write {file}` : also works with visual selection
55     - visually select a chunk
56     - `write! resources/xwrite`
57         - we are still at vim.md, with the visually selected piped to the content of xwrite
58 - `sav {file}` / `saveas {file}` : save the current buffer to file and switch to that file, doesn't
59   work with visual selection
60     - `sav resources/saveas`
61         - we are now at resources/saveas
62 - `r {file}` / `read {file}` : paste the content of {file} here
63     - `read resources/xwrite`
64 - `r! {command}` : run shell command {command}, paste the output here
65     - `r!ls -al`
